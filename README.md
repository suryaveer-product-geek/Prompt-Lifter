## 🚀 Overview

Prompt Lifter is a Chrome extension designed to help users generate **well-structured, high-quality prompts** for AI tools.

Instead of struggling to phrase the “perfect prompt,” users can simply dump their thoughts — and the extension transforms them into **clear, targeted, and effective AI prompts**.

## 🎯 Problem

Most users interacting with AI tools:

* Write vague or unstructured prompts
* Miss critical context
* Receive generic or low-quality responses

The core issue:

> Users know *what they want*, but don’t know *how to ask for it*.

This leads to:

* Multiple retries
* Frustration with AI outputs
* Underutilization of AI capabilities

---

## 💡 Solution

Prompt Lifter acts as an **AI prompt optimizer layer**.

It allows users to:

* Freely write their raw thoughts
* Answer a few guided clarifying questions
* Automatically generate a **structured, context-rich prompt**

The result:

> Better inputs → significantly better outputs

---

## 🔁 Workflow Transformation

### Before

Think → Write vague prompt → Get generic response → Retry

### After

Think → Dump thoughts → Guided refinement → Get high-quality response

---

## ⚙️ How It Works

1. User writes a rough idea or question
2. Extension asks contextual follow-up questions
3. Collects missing details (intent, audience, constraints, format)
4. Generates a refined, structured prompt
5. User pastes it into any AI tool
